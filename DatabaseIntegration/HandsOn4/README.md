# Hands-On 4 – Query Optimization

Project: Vehicle Service Management System

Tasks Completed:

* Analyzed query execution plan using EXPLAIN FORMAT=JSON.
* Identified table scans in the query plan.
* Created indexes on vehicle_name and service_type columns.
* Verified indexes using SHOW INDEX.
* Compared execution plans before and after indexing.
* Demonstrated the N+1 query problem.
* Implemented JOIN-based solution to reduce database queries.

Tools Used:

* MySQL Command Line Client
* Python
* VS Code
